Navigation Bars

Having easy-to-use navigation is important for any web site.

With CSS you can transform boring HTML menus into good-looking navigation bars.

Navigation Bar = List of Links

A navigation bar needs standard HTML as a base

In our examples we will build the navigation bar from a standard HTML list.

A navigation bar is basically a list of link, so using the <ul> and <li> elements makes perfect sense:

Example explained:

The code in the exaple above is the standard code used in both vertical, and horizontal navigation bars, which you will learn more about int he next chapters.

Example explained:

  • diplay: block;- Displaying the links as block elements makes the whole link area clickable (not just the text), and it allows us to specify the width (and padding, margin, height, etc. if you want.)
  • width: 60px;- Block elements take up the full width available by default. We want to specify a 60 pixels width

You can also set the width of ul, and remove the width of anchor, as they will take up the full width available when displayed as block elements. This will produce the same result as our previous examples.
