  • float: left; - Use float to get block elements to float next to each other.
  • display:block; - Allows us to specify padding (and heifht, width, width, margins, etc. if you want).
  • padding: 8px; - Specify some padding between each <a> element, to make them look good.
  • background-color: white; - Add a background- color to each <a> element.

Tip: Add the background-color to <ul> instead of each <a> element if you want a full- width background color.

overflow: hidden

floated elements are taken out of the normal document flow and do not take up space. Because the children take up no space.

Adding overflow: hidden; triggers a new block formatting context tant prevent NavBar from "collapsing" when it has floated children.
